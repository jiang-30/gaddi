## version

1. `changeset pre enter <alpha/beta>`

2. `changeset` 选择要发布的包

3. `changeset version` 修改发布包的版本

4. `changeset publish` 发布包到仓库

    `npm whoami`    `npm login`    `npm config get registry`

5. `npx changeset pre exit`
